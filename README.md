# 🌡️ Temperature Converter

A simple and responsive **Temperature Converter** web application built using **HTML5, CSS3, and JavaScript**. It allows users to seamlessly convert temperature values between **Celsius and Fahrenheit** with real-time results and a clean user interface.

---

## 🚀 Features

- 🔄 Convert between Celsius and Fahrenheit
- ⚡ Real-time conversion with instant results  
- 📱 Fully responsive design  
- 🎨 Clean and minimal user interface  
- 🧩 Beginner-friendly JavaScript project  

---

## 🛠️ Tech Stack

- **HTML5** – Application structure  
- **CSS3** – Styling and responsiveness  
- **JavaScript (ES6)** – Conversion logic and DOM manipulation  

---

## ⚙️ How It Works

1. Enter a temperature value  
2. Select the desired unit (Celsius, Fahrenheit, or Kelvin)  
3. The application processes the input using JavaScript  
4. Converted values are displayed instantly
